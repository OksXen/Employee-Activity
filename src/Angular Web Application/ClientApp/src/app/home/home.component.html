Angular, .NET Core, ORM and Azure (continous integration)

There are 3 applications in the Visual Studio 2019 solution.

  • .NET Core Angular Web Application
  • .NET Core Library
  • xUnit Test

.NET Core Angular Web Application

The angular client app runs within the .NET Core Angular Web application. The .NET Core Web Application has REST API that angular client app use to send and retrieve data.

.NET Core Library

The codes in the library can be reused and shared with one or more websites. The library uses Entity Framework to manage insert and delete data in MS SQL Server. It is also responsible for receiving and displaying data back to Angular.

The list is cached for an hour, but is 'touched' when there a new employee activity is added to a table.
